## About
调整 Tank 对倒地的幸存者造成的伤害。

## Dependencies
- [SourceScramble](https://github.com/nosoop/SMExt-SourceScramble)

## Notes
该插件只支持合作模式。

## ConVar
```c
// Easy, Normal, Hard 难度的伤害
// 75.0=游戏默认
l4d2_tank_pound_damage_normal "50.0"

// impossible 难度的伤害
// 150.0=游戏默认
l4d2_tank_pound_damage_impossible "150.0"
```
