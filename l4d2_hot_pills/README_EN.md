## About
Pills heal over time.

## ConVar
```c
// The healing interval.
l4d2_hot_pills_interval "0.1"

// How much health is healed each time.
l4d2_hot_pills_increment "2.0"

// Maximum Heal Health.
l4d2_hot_pills_max_heal "50.0"
```
