## About
Provided give ammo command.

## Dependencies
- [multicolors](https://github.com/fdxx/l4d2_plugins/tree/main/multicolors) 

## ConVar
```c
// Use the interval between command. -1.0=Disable the plugin.
l4d2_give_ammo_time "90.0"
```

## Command
```c
// Client command, give ammo.
sm_ammo
```
