## About
在地图上产生多个 Witch

## Dependencies
- left4dhooks

## ConVar
```c
// 限制活着的 Witch 数量。
l4d2_multi_witches_limit "40"

// Witch 产生时间。
l4d2_multi_witches_spawn_time "30.0"

// 超过这个距离的 Witch 将会被自动杀死。
l4d2_multi_witches_kill_distance "1800.0"
```
